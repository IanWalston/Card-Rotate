Rotation Animation using CSS transform.

There are three types of rotation available to us using the css property, transform. These types correspond with the three axes in 3D space: X, Y, and Z.

CSS transformations happen instantly, as soon as the class is added. To make the transition smooth, you can give the div the css property transition an amount of seconds equal to the duration of the animation, like this: 

div {
  transition: 1s;
  padding: 6px;
}
Click each card to test out the animation.

RotateZ(180deg)

.rotz180 { transform: rotateZ(180deg); }

RotateY(180deg)

.rotz180 { transform: rotateY(180deg); }

RotateX(180deg)

.rotz180 { transform: rotateX(180deg); }

X and Y

.rotz180 { transform: rotateX(180deg) rotateY(180deg); }

X and Z

.rotz180 { transform: rotateX(180deg) rotateZ(180deg); }

Y and Z

.rotz180 { transform: rotateY(180deg) rotateZ(180deg);}

X, Y and Z

.rotz180 { transform: rotateX(180deg) rotateY(180deg)
          rotateZ(180deg);}
